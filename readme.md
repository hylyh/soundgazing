# self-care-jam

## how to do

1. `npm install -g gulp jspm`
2. `npm install`
3. `jspm install`
4. `gulp`

## how to build
1. `gulp build`

## how to deploy
1. `gulp build`
2. `git add dist`
3. `git commit -m "Deploying to gh-pages"`
4. `git subtree push --prefix dist origin gh-pages`

or, `./gh-pages-push.sh`
